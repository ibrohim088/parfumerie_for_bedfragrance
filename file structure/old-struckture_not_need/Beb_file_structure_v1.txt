  [Telegram Bot]
       │
       ├── /start bosganda → Inline tugma: "🌐 Saytni Ochish"
       │                      ↓
       │                   [Telegram WebApp]
       │                      ↓
       │               Next.js sayt ochiladi
       │               (to'liq brauzer ichida)
       │
       └── Bot auth jarayoni:
           1. User raqamini ulashadi (Telegram contact)
           2. Bot backend ga /auth/send-otp yuboradi
           3. SMS orqali OTP keladi
           4. User OTP kiritadi
           5. Backend JWT qaytaradi
           6. Bot session'da token saqlanadi

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Saytda:                          Botda:
  ─────────────────────────────    ─────────────────────────────
  1. +998 XX XXX XX XX kiriting    1. "📱 Raqam ulashish" tugmasi
  2. SMS OTP keladi                2. Telegram raqamni o'zi beradi
  3. 6 raqam kiritiladi            3. SMS OTP keladi
  4. JWT token → cookie'ga         4. 6 raqam kiritiladi
                                   5. JWT → bot session'ga

  Forgot Password:
  ─────────────────────────────
  1. "Parolni unutdim" bosiladi
  2. Telefon raqam kiritiladi
  3. SMS OTP yuboriladi
  4. OTP + yangi parol o'rnatiladi
